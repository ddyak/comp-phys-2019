## Requirements:

* **g++ compiler** v9.1 (Or any other for supporting C++17).
* **gnuplot** v5.2

## Compile:

```console
cd solutions
makefile build
```

## Execute:

```console
./task2                             # execute code
gnuplot convergenceRate.gdi         # compile figure
xdg-open convergenceRate.pdf        # show pdf using standard utility
```
